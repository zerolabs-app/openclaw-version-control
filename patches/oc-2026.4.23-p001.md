# OpenClaw Patch oc-2026.4.23-p001

## Name

Recurring Task Runtime Workspace Override Patch

## Applies To

- Mission Control: 1.0.0
- OpenClaw: 2026.4.23

## Purpose

- Move recurring task behavior to runtime workspace overrides
- Allow recurring-task.md to be patched without rebuilding Mission Control
- Establish workspace runtime prompt override architecture
- Keep bundled prompts as fallback-only behavior
- Preserve workspace artifact requirements for recurring jobs

## Runtime Override Location

Operational helper prompts live in:

~/.openclaw/workspace/mission-control/prompts/

## Runtime Behavior

Mission Control should:

1. First check the runtime workspace prompt location
2. Use the workspace override if present
3. Fallback to bundled prompts only if no override exists

## File To Update

~/.openclaw/workspace/mission-control/prompts/recurring-task.md

## Change Summary

- Add runtime workspace override support
- Move recurring-task.md into workspace-managed runtime prompts
- Add Workspace Defaults Awareness Rule
- Add Delivery Independence Rule
- Add Cron Delivery Inheritance Rule
- Add Direct Response Prevention Rule
